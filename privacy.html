Privacy Policy

Last updated: May 13, 2026

What RoPairly Does

RoPairly lets two linked Roblox users save and view a shared list of Roblox games.

Information We Use

  • Roblox user ID and username for account detection and linking.
  • Shared saved game details such as title, place ID, image URL, rating, player count, genre, and Roblox game URL.
  • Pairing, verification, trusted-device, presence, and shared activity records needed for RoPairly features.
  • Extension settings stored locally in Chrome storage.

How Information Is Stored

RoPairly stores extension settings and trusted-device data in Chrome local storage. Shared RoPairly data is stored in Supabase and protected by database access rules.

What We Do Not Collect

  • RoPairly does not read Roblox passwords.
  • RoPairly does not access Roblox cookies or .ROBLOSECURITY tokens.
  • RoPairly does not sell user data.

Third-Party Services

RoPairly uses Supabase for authentication, database storage, and verification functions. RoPairly also calls public Roblox APIs to detect accounts and load game/avatar-related public data.

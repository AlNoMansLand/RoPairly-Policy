RoPairly Privacy Policy

Last updated: May 13, 2026

Overview

RoPairly is a Chrome extension that lets two linked Roblox users save and view a shared list of Roblox games. This policy explains what information RoPairly uses and how it is handled.

Information RoPairly Uses

  • Roblox user ID and username for account detection, verification, recovery, and partner linking.
  • Saved Roblox game details, including title, place ID, image URL, game URL, rating, player count, and genre.
  • Pairing data, verification requests, trusted-device records, partner presence status, and shared activity history.
  • Extension preferences stored in Chrome local storage, such as display and visibility settings.

How Information Is Stored

RoPairly stores extension preferences and trusted-device data in Chrome local storage on your browser. Shared RoPairly data is stored in Supabase and is protected by server-side database access rules.

How Information Is Used

  • To verify that a Roblox account belongs to the current user.
  • To link two RoPairly users into one shared space.
  • To save, display, and sync shared Roblox games.
  • To show optional partner presence and shared activity information.

What RoPairly Does Not Do

  • RoPairly does not read Roblox passwords.
  • RoPairly does not access Roblox cookies or .ROBLOSECURITY tokens.
  • RoPairly does not sell user data.
  • RoPairly does not use remote JavaScript in the extension.

Third-Party Services

RoPairly uses Supabase for authentication, database storage, and verification functions. RoPairly also calls public Roblox APIs to detect accounts and load public game information.
